# Semantic Web Big HW1 — Recipe Recommender

## Project Description

This project is a Java Spring Boot web application developed for the Semantic Web homework assignment.

The application recommends suitable recipes to users based on their cooking skill level and preferred cuisine type. The project uses XML as the main data storage format and applies Semantic Web/XML technologies such as XSD, XSL, XPath, and XQuery-style XML filtering.

The application allows users to:
- view all recipes stored in the XML file;
- add new recipes through a web form;
- add new users through a web form;
- select an active user without using email or password authentication;
- recommend recipes based on cooking skill level;
- recommend recipes based on both cooking skill level and preferred cuisine;
- filter recipes by cuisine type;
- view the full details of a selected recipe;
- display recipes through XSL formatting, where recipes matching the selected user's skill level are highlighted in yellow and the other recipes are highlighted in green.
